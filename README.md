# Sistema de Gestión de Tareas Personal

Un sistema simple y eficiente desarrollado en Python para gestionar tareas diarias desde la terminal.

## Características Principales

- ✅ **Agregar tareas** con título, descripción y fecha
- ✅ **Ver tareas pendientes** filtradas por estado
- ✅ **Marcar tareas como completadas**
- ✅ **Persistencia de datos** en archivo JSON
- ✅ **Sin dependencias externas** (solo librerías estándar de Python)
